# Dock Video Player

## 简介

利用Dock图标动态替换API，实现的一款在Mac Dock栏播放视频的程序。

## 使用教程

1. 运行`npm install`安装依赖（ffmpeg-static安装可能会有点问题，建议直接使用国内镜像源下载）
2. 运行`npm start`启动Electron应用
3. 加载视频
4. 点击播放即可在Dock栏中看到视频画面
5. 仓库中提供了BadApple的范例视频，建议将音频的播放时间设置为1倍速+提前0.2秒播放，体验最佳
